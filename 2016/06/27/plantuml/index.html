概念

PlantUML is a open-source tool that allows to quickly write :

  • Sequence diagram,
  • Usecase diagram,
  • Class diagram,
  • Activity diagram, (here is the new syntax),
  • Component diagram,
  • State diagram,
  • Deployment diagram,
  • Object diagram.
  • wireframe graphical interface
